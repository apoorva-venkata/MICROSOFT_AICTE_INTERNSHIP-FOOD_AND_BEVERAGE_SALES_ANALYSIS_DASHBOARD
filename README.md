# 📊 Food and Beverage Sales Analysis Dashboard

## 📌 Project Overview

The Food and Beverage Sales Analysis Dashboard is developed using Microsoft Power BI to analyze sales performance and business trends in the food and beverage sector.

The dashboard transforms raw sales data into meaningful insights that help understand revenue patterns, product performance, and customer purchasing behavior.

This project demonstrates the use of data analytics, visualization techniques, and DAX measures to convert transactional data into actionable business insights.

---

## 🛠 Tools and Technologies

- 📊 Microsoft Power BI
- 🧮 DAX (Data Analysis Expressions)
- 📑 Microsoft Excel
- 📉 Data Visualization Techniques

---

## 📂 Dataset

The analysis is based on sales transaction data containing information such as:

- 🛍 Product details  
- 📦 Order information  
- 🔢 Quantity and price  
- 💰 Sales revenue  

The datasets used in this project are available in the **datasets folder** of this repository.

---

## 📊 Key Metrics Analyzed

The dashboard focuses on several important business metrics:

- 💰 Total Revenue – Overall sales generated from products  
- 🧾 Total Orders – Number of customer transactions  
- 📊 Average Ticket Price (ATP) – Average revenue per order  
- 📦 Product Performance – Sales comparison across products  
- 🧩 Sales Distribution – Understanding revenue contribution by category  

---

## 📈 Dashboard Features

The Power BI dashboard includes interactive visualizations such as:

- 📉 Revenue trend analysis  
- 🛍 Product-wise sales performance  
- 📦 Order distribution insights  
- 📊 Key Performance Indicator (KPI) cards  

These visuals help stakeholders quickly understand business performance and identify trends.
